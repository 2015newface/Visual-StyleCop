# Visual StyleCop
Copy of StyleCop (Andy Reeves) from stylecop.codeplex.com for visual studio 2015 without resharper

I am cleaning the solution and it should be easier to work than previous version.
No command lines only visual studio 2015 and GitHub.

The wiki is in progress and sometimes you can browse to a broken link, please be patient.

Current published version : 4.7.51 on [Visual Studio Extension](https://visualstudiogallery.msdn.microsoft.com/cac2a05b-6eb6-4fa2-95b9-1f8d011e6cae)
